# UltimateAuth Authentication EntityFrameworkCore

Entity Framework Core persistence for UltimateAuth authentication.

## Purpose

Provides durable storage for:

- Authentication state
- Session lifecycle
- Identity resolution

## Features

- Persistent authentication state
- Scalable session storage
- Production-ready infrastructure

## Notes

- Requires EF Core setup
- Migrations handled by application

## When to use

- Production environments
- Distributed systems

## Alternatives

- CodeBeam.UltimateAuth.Authentication.InMemory
- Custom packages
