# UltimateAuth Authorization EntityFrameworkCore

Entity Framework Core persistence implementation for the UltimateAuth Authorization module.

## Purpose

Provides durable storage for:

- Roles
- Claims
- Authorization rules

## Features

- Persistent role storage
- Claims-based authorization support
- Scalable architecture

## Notes

- Requires EF Core setup
- Migrations handled by application

## When to use

- Production environments

## Alternatives

- CodeBeam.UltimateAuth.Authorization.InMemory
- Custom packages
