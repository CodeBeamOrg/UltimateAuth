# UltimateAuth Policies

Policy evaluation module for UltimateAuth.

## Purpose

This package provides:

- Authorization policy evaluation logic
- Permission-based decision making
- Reusable policy helpers

## Does NOT include

- Persistence
- User or credential management
- ASP.NET Core integration

⚠️ This package is typically installed transitively via:

- CodeBeam.UltimateAuth.Server

In most cases, you do not need to install it directly unless you are building custom policy logic.
