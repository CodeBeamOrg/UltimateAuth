# UltimateAuth Users EntityFrameworkCore

Entity Framework Core persistence implementation for the UltimateAuth Users module.

## Purpose

This package provides a production-ready user storage implementation using EF Core.

## Features

- Persistent user storage
- Relational database support
- EF Core integration

## Notes

- Requires EF Core setup
- Migrations must be handled by the application

## When to use

- Production environments
- Applications requiring persistent storage

## Alternatives

- CodeBeam.UltimateAuth.Users.InMemory (development/testing)
- Custom persistence implementations
